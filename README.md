## Instructions
##### 1 - Download or clone the project.
##### 2 - Change in src/main.js the value for Vue.http.options.root for your server name
##### 3 - Run npm install
##### 4 - Run npm run dev
##### 5 - by default the application will be on http://localhost:8080
